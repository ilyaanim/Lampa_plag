# Lampa Wide Covers Plugin

Плагин для [Lampa.mx](https://lampa.mx), который заменяет стандартные вертикальные постеры на широкие обложки (backdrop) формата 16:9.

## Скриншот

| До (стандарт) | После (широкие) |
|---|---|
| Постеры 2:3 | Backdrop 16:9 |

## Установка

1. Откройте **Lampa** → **Настройки** → **Расширения** → **Добавить плагин**
2. Вставьте URL плагина:
   ```
   https://YOUR_HOST/wide_covers.js
   ```
3. Перезапустите Lampa

### Локальная установка (для разработки)

Разместите файл `wide_covers.js` на любом HTTP-сервере и используйте его URL.

## Использование

После установки плагин активен по умолчанию.

Для управления:
- **Настройки** → **Интерфейс** → **Широкие обложки** → Включено/Выключено

## Что делает плагин

- Заменяет вертикальные постеры (poster_path) на горизонтальные обложки (backdrop_path) из TMDB
- Изменяет размер карточек для корректного отображения в формате 16:9
- Если у фильма нет backdrop-изображения, остаётся стандартный постер
- Не затрагивает карточки типа `card--wide` и `card--collection` (они уже широкие)

## Совместимость

- Lampa 3.x
- Все платформы: Web, Android, WebOS, Tizen
